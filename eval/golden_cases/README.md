# Golden Cases

이 디렉토리는 Luddite v0 평가용 golden case를 저장한다.

초기에는 최신 PPT 8개 중 3개를 JSON으로 넣고, 나머지는 Codex가 parser 구현 후 추가한다.

현재 포함:

```text
golden_pawnshop_f88.json
golden_coca_cola_ambani.json
golden_laser_drone.json
```

추가 예정:

```text
golden_shareholder_people.json
golden_uk_chaos.json
golden_us_china_summit.json
golden_teachers_day.json
golden_summer_shorts.json
```
