## 5. jibi scoring에 반영할 규칙

### 5.1 점수 가산

```text
+ 이상징후가 명확함
+ 3단 이상 확장 가능
+ 기존 슈카월드 래퍼토리와 연결
+ 공식자료/통계/그래프 확보 가능
+ 한국 시청자 체감 또는 내부 punchline 가능
+ 제목만 봐도 “엥?”이 생김
```

### 5.2 감점/보류

```text
- 특정 기업 홍보/비판으로 보임
- 너무 따끈따끈해서 검증 어려움
- 서브급 단발 사건
- 패턴이 너무 뻔함
- 민감 집단/정치/국제분쟁 리스크
- 이미지 저작권 위험이 큼
```

### 5.3 label 처리

`주제 찾기`의 label은 이렇게 다루는 것이 안전하다.

```text
제작 O / 방송 O      → positive
제작 O / 방송 X      → produced_but_rejected
제작 O / 방송 blank  → pending_or_unknown
제작 X               → rejected_or_not_pursued
blank                → unlabeled
```
