## 2. 주요 패턴

```text
- 16:9 wide
- 흰 배경 중심
- 맑은 고딕 계열
- 본문 28pt 중심
- 한 장에 한 메시지
- section title로 호흡 조절
- speaker notes에 [내용]/[이미지] 출처 보존
- 이미지/GPT 생성/기사 캡처/차트가 혼합됨
```

## 4. Golden set별 핵심 문법

### 전당포 주식회사

```text
해외 낯선 회사 → 한국식 배경 설명 → 신흥국 금융 구조 → 창업자/규제 리스크
```

### 코카콜라를 이기는 방법

```text
코카인 하마 → 암바니 → 릴라이언스식 시장 교란 → 캄파콜라 가격전쟁
```

### 요즘 뜨는 레이저 치료

```text
드론 비용 비대칭 → 레이저 무기 → 한국 천광 → 피부과 레이저 punchline
```

### 여름에 회사에서 반바지

```text
폭염 체감 → 일본 40도 용어/경보 → 에너지/복장 규범 → 한국 회사 반바지
```

### 슈승님의 은혜

```text
스승의 날 내부 이벤트 → 카네이션 품귀 → 청탁금지법 → 교권/민원 변화
```

### 국민도 주주가 되는가

```text
시장 급락 → 정책 발언 원문 분해 → AI/반도체 구조론 → 국민배당/투자자 리스크
```

### 대혼돈의 영국

```text
지방선거 충격 → 양당 붕괴 → 개혁당 부상 → 러스트벨트/지역격차 → 시장 리스크
```

### 미중 정상회담

```text
정상회담 → 2017년 기억 → 트럼프 국내 위기 → 중국 도움 요청 → 기업별 수혜자
```

## 5. piti가 배워야 할 것

```text
- PPT는 문서가 아니라 장면의 연속이다.
- 출처는 본문보다 notes에 보존한다.
- 긴 원문은 여러 장으로 쪼갠다.
- quote slide, explainer slide, punchline slide는 문장 밀도가 다르다.
- 대형 덱은 section title이 필수다.
- 이미지가 없는 텍스트 슬라이드도 충분히 많다.
```
