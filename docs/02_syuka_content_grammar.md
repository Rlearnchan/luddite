# 02. Syuka Content Grammar

작성일: 2026-05-16
상태: v0.1 draft

## 1. 목적

이 문서는 Luddite의 핵심 판단 체계다. `jibi`가 어떤 seed를 골라야 하는지, `anny`가 어떤 구조로 풀어야 하는지, `piti`가 어떤 장면 단위로 나눠야 하는지를 정리한다.

슈카월드 콘텐츠의 핵심은 정보량 자체가 아니라 **이상한 seed를 구조적 이야기로 키우는 능력**이다.

## 2. 한 줄 공식

```text
엥? 하는 seed
→ 왜 지금 화제인지 숫자/사건으로 증명
→ 시청자가 이해할 수 있는 배경 설명
→ 더 큰 구조 문제로 확장
→ 한국/내부/밈으로 회수
→ 마지막 질문 또는 찝찝한 결론
```

## 3. 좋은 seed의 조건

좋은 seed는 다음 조건 중 여러 개를 만족한다.

| 조건 | 설명 | 예시 |
|---|---|---|
| 즉각적인 의문 | 제목만 봐도 “왜?”가 생김 | 베트남 전당포가 상장한다고? |
| 숫자 증명 | 성장률, 가격, 득표율, 점유율 등으로 충격을 보여줌 | F88 매출 10배, 영국 노동당 17% |
| 배경 설명 가능 | 한국 시청자가 이해할 수 있는 번역 지점이 있음 | 한국 전당포, 직장 복장, 스승의 날 |
| 3단 이상 확장 | 기사 하나에서 구조적 문제로 확장 가능 | 하마 → 암바니 → 캄파콜라 |
| 한국 연결 | 국내 사례, 회사 내부 농담, 한국 산업으로 회수 가능 | 레이저 무기 → 피부과 레이저 |
| 출처 깊이 | seed 기사 외에 공식자료/통계/후속자료 확보 가능 | World Bank, FT, Reuters, 정부자료 |
| punchline 가능 | 마지막에 질문/농담/반전으로 닫을 수 있음 | “슈카콜라가 경쟁할 날?” |

## 4. 반복되는 story archetype

### 4.1 해외 낯선 회사 → 한국식 번역 → 구조적 시장

대표 샘플: `전당포 주식회사`

```text
베트남 F88 상장 추진
→ 매출/점포/순이익 급성장
→ 그런데 회사가 전당포다
→ 한국의 전당포 기억 설명
→ 베트남 신용시장과 비공식 고용
→ 오토바이 담보대출
→ 창업자 서사와 추심 리스크
```

사용 조건:

- 낯선 해외 회사가 있다.
- 성장 숫자가 있다.
- 한국 시청자가 이해할 수 있는 유사 제도나 기억이 있다.
- 단순 성공담이 아니라 그림자/리스크가 있다.

나쁜 사용:

- 특정 기업 홍보로 끝나는 경우
- 성장 숫자만 있고 사회적 구조가 없는 경우
- 한국 연결이 억지인 경우

### 4.2 A로 시작해서 사실은 B 이야기

대표 샘플: `코카콜라를 이기는 방법`

```text
콜롬비아 코카인 하마 인도행
→ 파블로 에스코바르의 하마 문제
→ 안락사 논란
→ 암바니 가문 등장
→ 릴라이언스식 가격 교란
→ 캄파콜라와 코카콜라/펩시 경쟁
→ 슈카콜라 punchline
```

사용 조건:

- seed 자체가 기묘하고 눈길을 끈다.
- 중간에 큰 인물/기업/산업으로 넘어갈 수 있다.
- 처음 소재가 마지막에 회수된다.

### 4.3 생활 체감 → 해외 사례 → 제도/문화 변화 → 내부 회수

대표 샘플: `여름에 회사에서 반바지 입어도 되나요`

```text
5월인데 40도에 육박
→ 한국/일본 조기 폭염
→ 일본의 40도 용어와 경보
→ 에너지 가격과 오피스 복장
→ 남성 반바지 논란
→ 한국 기업/공무원 쿨비즈
→ 슈카친구들 반바지 허용 요청
```

사용 조건:

- 시청자가 바로 체감한다.
- 해외에서도 비슷하거나 더 극단적인 사례가 있다.
- 문화 규범이나 산업 변화로 연결된다.
- 내부 농담으로 회수 가능하다.

### 4.4 기술 비용 역전 → 국가 경쟁 → 한국 연결 → 말장난 회수

대표 샘플: `요즘 뜨는 레이저 치료`

```text
싼 드론을 비싼 미사일로 막는 비효율
→ EMP/샷건/그물총 등 대안
→ 레이저 무기
→ 아이언빔/헬리오스/드래곤파이어/천광
→ 한국의 방산 레이저
→ 한국의 피부과 레이저 치료
```

사용 조건:

- 비용 역전이 명확하다.
- 각국 경쟁 구도가 있다.
- 한국 기술/산업 연결이 가능하다.
- 제목 말장난이 마지막에 회수된다.

### 4.5 정책 발언 → 시장 반응 → 원문 분해 → 투자자 관점 질문

대표 샘플: `국민도 주주가 되는가`

```text
코스피 급등락
→ 국민배당금 논란
→ 김용범 원문 문장별 해석
→ AI/반도체/국가전략
→ 국민성장펀드
→ 원금손실/환매제한/정부 손실부담
→ 국민이 주주가 된다는 의미 질문
```

사용 조건:

- 정책 발언이 시장에 영향을 준다.
- 원문이 길고 오해 가능성이 있다.
- 문장별 분해가 유효하다.
- 시청자의 투자자 관점으로 되물을 수 있다.

### 4.6 정치 이벤트 → 기존 질서 붕괴 → 경제/지역 구조 → 시장 리스크

대표 샘플: `대혼돈의 영국`

```text
영국 지방선거 참패
→ 노동당/보수당 동시 붕괴
→ 개혁당 부상
→ 스타머 사퇴 압박
→ 러스트벨트/런던 격차
→ 양당제 붕괴
→ 국채금리와 정치 리스크
```

사용 조건:

- 선거 결과가 단순 승패를 넘어 체제 균열을 보여준다.
- 지역/계층/산업 구조가 있다.
- 미국, 한국 등 다른 나라의 기시감과 연결된다.
- 금융시장 또는 정책 리스크까지 확장된다.

### 4.7 지정학 이벤트 프리퀄

대표 샘플: `미중 정상회담`

```text
미중 정상회담 개최
→ 2017년 방중 짤 회상
→ 9년 뒤 권력관계 변화
→ 트럼프 국내 위기와 인플레
→ 이란 전쟁과 유가
→ 중국에 도움 요청
→ 의전/이름 해프닝
→ 보잉/테슬라/엔비디아 수혜자 분석
```

사용 조건:

- 이벤트 자체보다 이벤트 전의 사정이 더 재미있다.
- 과거 장면과 현재 권력관계의 대비가 있다.
- 기업/시장 수혜자 분석으로 이어진다.

### 4.8 내부 밈 → 사회 변화

대표 샘플: `슈승님의 은혜`

```text
스승의 날 사내 이벤트
→ 슈카쌤 내부 농담
→ 카네이션 구하기 어려움
→ 꽃집 기사
→ 청탁금지법
→ 교권/민원 변화
→ 교육현장 자정 노력
```

사용 조건:

- 내부 농담이 도입으로 작동한다.
- 10장 이내에 외부 사회 현상으로 연결된다.
- 단순 자기 얘기로 끝나지 않는다.

## 5. 도입부 hook 유형

| 유형 | 특징 | 예시 문장 |
|---|---|---|
| 엥?형 | 상식과 어긋나는 소재 | “최근 상장에 도전하는 금융 회사가 있다. 바로 베트남의 전당포” |
| 체감형 | 날씨, 직장, 생활 이슈 | “와 요즘 날씨 실화인가요?” |
| 짤 회상형 | 과거 유명 장면을 소환 | “당시엔 엄청난 짤이 나오며 화제가 되었는데” |
| 숫자 충격형 | 급락, 급등, 점유율 | “전국 득표율 추정치가 17%까지 하락” |
| 내부 농담형 | 회사/대표/직원 밈 | “귀여운 슈친 직원들” |
| 말장난형 | 제목의 중의성 | “요즘 뜨는 레이저 치료” |

## 6. 중간 확장 방식

좋은 중간부는 seed를 반복 설명하지 않고, 관점을 바꾼다.

```text
사건 → 숫자
숫자 → 과거 사례
과거 사례 → 제도
제도 → 산업
산업 → 사람
사람 → 리스크
리스크 → 질문
```

## 7. 한국 연결 방식

한국 연결은 억지로 “한국도 그렇다”라고 말하는 것이 아니다. 다음 중 하나가 있어야 한다.

- 한국에 비슷한 제도가 있었다.
- 한국 시청자가 가진 기억이 있다.
- 한국 산업/기업이 실제로 연결된다.
- 한국의 문화나 직장생활로 체감된다.
- 슈카친구들 내부 상황과 자연스럽게 이어진다.

## 8. 결론 방식

슈카월드식 결론은 단정적 결론보다 질문형/찝찝함이 많다.

좋은 결론:

```text
- 이 시스템은 과연 양지화될 수 있을까?
- 국민이 주주가 된다는 건 배당만 받는다는 뜻일까?
- 양당제가 무너지면 시장은 무엇을 가격에 반영할까?
- 슈카콜라가 캄파콜라와 경쟁하는 날이 올까?
```

나쁜 결론:

```text
- 따라서 이 기업은 유망하다.
- 이 정책은 무조건 성공한다.
- 이 나라는 망했다.
- 이 주식은 사야 한다.
```

## 9. 위험한 소재

리서치 인수인계에서 확인된 기피/주의 영역은 별도 보안 문서에 정리한다. 콘텐츠 문법 차원에서는 다음 위험을 표시해야 한다.

- 특정 민족/인종/종교 비판으로 읽힐 위험
- 특정 기업 홍보 또는 공격으로 보일 위험
- 선거/정치 고위험 이슈에서 검증되지 않은 단정
- 의료/건강 정보의 과도한 일반화
- 저작권 위험 이미지
- 구독 기사 전문 장문 인용
- SNS 계정 노출

## 10. Luddite 적용 원칙

- `jibi`는 seed가 어떤 archetype으로 확장될 수 있는지 기록한다.
- `anny`는 storyline 생성 시 최소 하나의 archetype을 명시한다.
- `piti`는 section title을 통해 archetype 전환을 시각화한다.
- 평가자는 결과물이 “기사 요약”이 아니라 “이야기 구조”가 되었는지 본다.
